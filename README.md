# mfm
memory fregment management. A tool save your brain memory.
记忆碎片管理，使用提示词，提取大量内容。
（上学时做的一个小工具，今天被翻出来了。）

功能：通过提示词，搜索文件。
	1、输入提示词后，回车搜索。
	2、结果列表中，单击表项为复制。
	3、双击表项为打开文件。
	4、右击三次为删除。
	5、使用#new#命令新建文件，格式请参考下文：“文件储存格式为：”。
	6、使用#directory#命令打开文件存储文件夹。
