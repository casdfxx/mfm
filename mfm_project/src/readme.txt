使用环境：WIN10，JDK18
功能：通过提示词，搜索文件。
	1、输入提示词后，回车搜索。
	2、结果列表中，单击表项为复制；
	3、双击表项为打开文件；
	4、右击三次为删除。
	
第一次使用时需要编译文件，
在freyaread，和ui同级目录下运行如下命令：
javac ui/UI.java
pause

第一次使用需要配置文件存储路径。在弹窗中选择。
系统环境变量有两个:
mfm:文件存储路径。
mfm_count:文件最大序号，自增。

运行方法：
	一、
	建议写成.bat，并添加到path中快捷调用：
	start javaw  ui/UI

	二、
	如果可以导出为.jar 运行方法：
	导出为mfm.jar后双击启动。或者使用bat方式：
	start javaw -jar mfm.jar



/**************注意**************/
若有备份文件，在第一次使用前，请将备份文件
放入想要存储的目录中。mfm_count会以当前
文件数做为基础。

文件储存格式为：
提示词1
提示词2
（空行）
正文行1（可显示的内容）
正文行2---n（需打开才能显示的内容）
/******************************/
